# 🏥 Hospital Management System

A full-stack web application using JSP, Servlets, and Oracle Database.

## 🚀 Features

* Patient Login & Registration
* Doctor Dashboard
* Appointment Booking
* Admin Panel
* CRUD Operations

## 🛠 Tech Stack

* Java (JSP & Servlets)
* Oracle Database
* HTML, CSS, JavaScript
